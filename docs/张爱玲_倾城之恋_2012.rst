第一炉香
--------

如果湘粤一带深目削颊的美人是糖醋排骨，上海女人就是粉蒸肉。

她听说过，有一个人逛了庐山回来，带了七八只坛子，里面装满了庐山驰名天下的白云，预
备随时放一点出来点缀他的花园。为了爱而结婚的人，不是和把云装在坛子里的人一样的傻
么！
