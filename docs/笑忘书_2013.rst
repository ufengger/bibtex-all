庐结西郊
--------

几年前，妻子不知从哪儿听说有一种“猝死症”，而且多于半夜两三点突发于青年男性身上，
于是回家吓唬我说，你晚上要注意呀！可别睡得太死就睡过去啦！那时我们的女儿还小，妻
子半夜醒来替她盖被，总要顺便把手伸到我嘴边试试还有气儿没有，弄得我哭笑不得。
