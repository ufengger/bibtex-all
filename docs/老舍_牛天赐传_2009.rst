三、子孙万代
------------

天下最有效的办法就是收买；自己吃肉，得让旁人至少啃点骨头，英雄的成功都仗着随手往
外扔骨头。自私的人得看准了肉而决定舍了骨头；骨头扔出去，自有自告奋勇愿意当狗的。
