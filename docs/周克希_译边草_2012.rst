古文修养还是要的
----------------

manifold 是个常用的数学名词，指一类外延很广的抽象空间。从字面上看，它是由 many
加后缀 “fold” 构成的。前辈数学家姜立夫先生把它译作“流形”，显然是从文天祥《正气歌》
里“天地有正气，杂然赋流形”的后半句脱胎而来。它在法文中的对应词是 variété ，原义
为“多样性”等等。

“流形”因“杂然”而得“多样”的神韵，又由一个“形”字坐实了空间的本义，堪称绝译。
